### COVID-19 Data Analysis

**Description:**  
This project involves exploring and analyzing global COVID-19 data using SQL. The analysis focuses on data retrieval, aggregation, transformation, and filtering to extract meaningful insights.

**Tools Used:**  
SQL  

**Key Insights:**

- Analyzed trends in COVID-19 cases and deaths across different countries and continents.
