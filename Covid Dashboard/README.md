# COVID-19 Tableau Dashboard

## Overview

This project involves creating a Tableau dashboard to visualize the global impact of COVID-19. The dashboard integrates key metrics such as total cases, new cases, total deaths, and population data to provide an intuitive and interactive way to explore trends and patterns in the pandemic data.

## Features

The Tableau dashboard includes the following:

- **Global Map Visualization**:
  - Displays COVID-19 cases and deaths across countries.
  - Interactive tooltips with key statistics for each region.

- **Time-Series Analysis**:
  - Line charts showing the trend of cases and deaths over time.
  - Comparisons of regions based on new cases and new deaths.

- **Key Metrics Summary**:
  - Total cases, deaths, and infection rates aggregated by continent and country.
  - Dynamic filtering by date range, continent, or specific countries.

- **Population Impact Insights**:
  - Visualization of infection rates as a percentage of the population.
  - Breakdown of high-risk regions with the highest death rates.

## Dataset

The dataset used for this project includes:

- **Location**: Country or region.
- **Date**: Recorded date of data.
- **Total Cases**: Cumulative confirmed cases.
- **New Cases**: Cases reported on a specific day.
- **Total Deaths**: Cumulative confirmed deaths.
- **Population**: Total population of the country/region.

## Objectives

The primary goals of this dashboard are:

1. To provide an intuitive and visually engaging way to analyze COVID-19 data.
2. To identify trends in infection and death rates across regions.
3. To facilitate comparisons between countries and continents.
4. To highlight key metrics such as death rates and infection rates.

## How to Use

1. Download the Tableau dashboard file (`Covid Dashboard.twb`).
2. Open the file in Tableau Desktop.
3. Connect the dashboard to the COVID-19 dataset (if required).
4. Explore the various visualizations and filters to gain insights.

## Tools

- **Tableau Desktop**: For creating and visualizing the dashboard.
- **SQL**: Used to pre-process and analyze the raw data.
- **COVID-19 Dataset**: Global statistics categorized by country and continent.

## Results

The Tableau dashboard provides:

- A comprehensive view of global COVID-19 trends.
- Insights into regions with the highest infection and death rates.
- Interactive tools for filtering and exploring specific data points.
